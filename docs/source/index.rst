Welcome to the o2r-meta documentation
=====================================

``o2r-meta`` is a collection of tools for extract-map-validate workflows as part of the `o2r service <https://o2r.info/architecture/>`_. The library is developed as part of the `DFG-funded <https://o2r.info/about/#funding>`_ research project Opening Reproducible Research `o2r <https://o2r.info>`_. Please report `Bugs <https://github.com/o2r-project/o2r-meta/issues>`_.
